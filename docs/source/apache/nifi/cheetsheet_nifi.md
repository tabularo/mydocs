# MyNiFi Cheet Sheet

## Managing Environments
| Action                             | Command             |
|------------------------------------|---------------------|
| Navigate to the NiFi bin directory | `cd ~/dev/nifi/bin` |
| Check the status of NiFi           | `./nifi.sh status`   |
| Start NiFi                         | `./nifi.sh start`|
| Stop Nifi | `./nifi.sh stop` |
